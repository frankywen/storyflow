## 一、配音/字幕系统

### 1.1 配音生成流程

```
场景对话提取 → 角色音色匹配 → TTS生成音频 → 音频预览 → 视频合成
```

**流程说明：**

1. 从故事解析结果提取每个场景的角色对话和旁白内容
2. 根据角色属性（性别/年龄）自动匹配音色，用户可手动调整
3. 调用TTS服务（云端API或Edge-TTS）生成音频文件
4. 用户可预览每个音频，不满意可重新生成或调整音色
5. 确认后将音频与视频合成，生成带配音和字幕的最终视频

**配音内容结构示例：**
```json
{
  "scene_id": 1,
  "dialogues": [
    {"character": "小明", "text": "今天天气真好！"},
    {"character": "小红", "text": "是啊，我们去公园吧。"}
  ],
  "narration": "小明和小红走在阳光明媚的街道上"
}
```

### 1.2 TTS服务接口设计

**统一接口：**
```go
type TTSProvider interface {
    GenerateVoice(text string, voiceID string) (*AudioResult, error)
    GetAvailableVoices() ([]Voice, error)
}

type AudioResult struct {
    AudioURL    string  // 生成的音频URL
    Duration    float64 // 音频时长（秒）
    VoiceID     string  // 使用的音色ID
}

type TTSConfig struct {
    Provider    string
    BaseURL     string
    APIKey      string
    AppID       string  // 部分服务需要
}
```

**支持的Provider：**

| Provider | 配置值 | 特点 |
|----------|--------|------|
| Edge-TTS | `edge-tts` | Microsoft免费服务、无需API Key、通过命令行调用 |
| 火山引擎 | `volcengine` | 国内稳定、按量计费、音色丰富 |
| 阿里云 | `alibaba` | 国内稳定、按量计费 |

**Edge-TTS说明：**
- Edge-TTS是Microsoft提供的免费TTS服务，通过Python库`edge-tts`调用
- 无需API Key，无需本地服务部署
- 安装方式：`pip install edge-tts`
- 调用方式：直接在Go中通过`exec.Command`调用edge-tts命令行工具

**配置示例：**
```env
# TTS Provider选择
TTS_PROVIDER=edge-tts  # edge-tts / volcengine / alibaba

# Edge-TTS命令路径（默认edge-tts，如已加入PATH可省略）
TTS_EDGE_CMD=edge-tts

# 火山引擎
TTS_VOLCENGINE_URL=https://openspeech.bytedance.com/api/v1/tts
TTS_VOLCENGINE_API_KEY=your-api-key
TTS_VOLCENGINE_APP_ID=your-app-id

# 阿里云
TTS_ALIBABA_URL=https://nls-gateway.cn-shanghai.aliyuncs.com/stream/v1/tts
TTS_ALIBABA_API_KEY=your-api-key
TTS_ALIBABA_APP_KEY=your-app-key
```

### 1.4 字幕生成系统

**字幕生成流程：**
```
配音音频 → 提取时间轴 → 生成字幕文件(SRT/VTT) → 视频合成时嵌入
```

**音频时间轴计算逻辑：**

字幕的start_time和end_time根据音频文件顺序累加计算：

```
场景1:
  - 对话1: 0.0s - 2.5s (duration=2.5s)
  - 对话2: 2.5s - 5.0s (duration=2.5s)
  - 旁白: 5.0s - 8.0s (duration=3.0s)
场景2:
  - 对话1: 8.0s - 10.0s (累加场景1总时长后开始)
  - ...
```

**计算代码示例：**
```go
func CalculateTimeAxis(audioFiles []AudioFile) []Subtitle {
    var subtitles []Subtitle
    currentTime := 0.0

    for _, audio := range audioFiles {
        subtitles = append(subtitles, Subtitle{
            Text:      audio.TextContent,
            StartTime: currentTime,
            EndTime:   currentTime + audio.Duration,
        })
        currentTime += audio.Duration
    }

    return subtitles
}
```

**字幕数据表：**
```sql
CREATE TABLE subtitles (
    id SERIAL PRIMARY KEY,
    story_id INT REFERENCES stories(id) ON DELETE CASCADE,
    scene_id INT REFERENCES scenes(id) ON DELETE CASCADE,
    subtitle_type VARCHAR(20) NOT NULL,    -- dialogue / narration
    character_id INT REFERENCES characters(id) ON DELETE SET NULL,
    text TEXT NOT NULL,
    start_time FLOAT NOT NULL,             -- 开始时间（秒）
    end_time FLOAT NOT NULL,               -- 结束时间（秒）
    style_config JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_subtitles_story ON subtitles(story_id);
CREATE INDEX idx_subtitles_scene ON subtitles(scene_id);
```

**字幕样式配置：**

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| position | bottom-center | 底部居中 |
| font_size | 24 | 字体大小(px) |
| font_color | #FFFFFF | 白色文字 |
| background_color | #00000080 | 半透明黑背景 |
| font_family | Arial | 字体 |

**style_config JSON结构：**
```json
{
  "position": "bottom-center",
  "font_size": 24,
  "font_color": "#FFFFFF",
  "background_color": "#00000080",
  "font_family": "Arial"
}
```

**字幕显示模式：**

| 模式 | 配置值 | 说明 |
|------|--------|------|
| dialogue_only | `dialogue_only` | 只显示角色对话 |
| narration_only | `narration_only` | 只显示旁白 |
| both | `both` | 对话+旁白都显示 |

### 1.6 音频与视频合成

**合成流程：**
```
无配音视频 → 导入配音音频 → 导入字幕 → ffmpeg合成 → 带配音字幕视频
```

**合成任务表：**
```sql
CREATE TABLE audio_video_synthesis (
    id SERIAL PRIMARY KEY,
    story_id INT REFERENCES stories(id) ON DELETE CASCADE,
    video_url VARCHAR(500),                -- 无配音视频URL
    audio_urls JSONB,                      -- 各场景音频URL列表
    subtitle_url VARCHAR(500),             -- 字幕文件URL
    output_url VARCHAR(500),               -- 合成后视频URL
    status VARCHAR(20) DEFAULT 'pending',  -- pending / processing / completed / failed
    progress INT DEFAULT 0,                -- 合成进度 0-100
    error_message TEXT,
    created_at TIMESTAMP DEFAULT NOW(),
    completed_at TIMESTAMP
);

CREATE INDEX idx_synthesis_story ON audio_video_synthesis(story_id);
```

**audio_urls JSONB结构：**
```json
[
  {"scene_id": 1, "audio_url": "https://.../scene1.mp3", "duration": 8.5},
  {"scene_id": 2, "audio_url": "https://.../scene2.mp3", "duration": 6.2},
  {"scene_id": 3, "audio_url": "https://.../scene3.mp3", "duration": 10.1}
]
```

**ffmpeg合成命令：**
```bash
# 合成视频+音频+字幕
ffmpeg -i input_video.mp4 \
       -i audio_track.mp3 \
       -vf "subtitles=subtitles.srt:force_style='Fontsize=24,PrimaryColour=&HFFFFFF'" \
       -c:v libx264 -c:a aac \
       output_with_audio_subs.mp4
```

**批量配音生成异步处理：**

配音生成采用异步任务处理，流程如下：

```
POST /api/v1/audio/generate
      ↓
创建任务记录，返回task_id
      ↓
后台goroutine逐场景生成音频
      ↓
GET /api/v1/audio/status/:task_id 查询进度
```

**任务状态表设计：**
```sql
CREATE TABLE audio_generation_tasks (
    id SERIAL PRIMARY KEY,
    story_id INT REFERENCES stories(id) ON DELETE CASCADE,
    status VARCHAR(20) DEFAULT 'pending',  -- pending / processing / completed / failed
    total_scenes INT DEFAULT 0,            -- 总场景数
    completed_scenes INT DEFAULT 0,        -- 已完成场景数
    progress INT DEFAULT 0,                -- 进度百分比 0-100
    error_message TEXT,
    created_at TIMESTAMP DEFAULT NOW(),
    completed_at TIMESTAMP
);
```

**并发控制：**
- 使用goroutine池控制并发数（默认3个并发）
- 每个音频生成完成后更新进度
- 失败重试机制：最多重试3次

### 2.4 验证码发送限制

防止滥用，设置发送频率限制：

| 限制项 | 规则 | 说明 |
|--------|------|------|
| 同邮箱发送间隔 | 60秒 | 同一邮箱60秒内只能发送一次 |
| 同邮箱每日上限 | 10次 | 同一邮箱每天最多10次 |
| 同IP每小时上限 | 20次 | 同一IP每小时最多20次 |

**验证逻辑：**
```go
func CanSendCode(email, ip string) bool {
    // 1. 检查60秒间隔
    lastSent := GetLastSentTime(email)
    if time.Since(lastSent) < 60*time.Second {
        return false
    }

    // 2. 检查每日上限
    todayCount := GetTodayCount(email)
    if todayCount >= 10 {
        return false
    }

    // 3. 检查IP每小时上限
    hourIPCount := GetHourIPCount(ip)
    if hourIPCount >= 20 {
        return false
    }

    return true
}
```

**数据隔离逻辑：**

所有API请求通过中间件注入user_id，查询时自动添加WHERE条件：

```go
// AuthMiddleware 解析JWT，注入user_id到context
func AuthMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        token := c.GetHeader("Authorization")
        claims := ParseJWT(token)
        c.Set("user_id", claims.UserID)
        c.Next()
    }
}

// 查询时自动过滤
func GetStories(userID int, filters map[string]string) []Story {
    query := db.Where("user_id = ?", userID)

    if status := filters["status"]; status != "" {
        query = query.Where("status = ?", status)
    }

    if search := filters["search"]; search != "" {
        query = query.Where("title ILIKE ?", "%"+search+"%")
    }

    return query.Find(&stories)
}
```
